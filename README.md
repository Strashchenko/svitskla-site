# СВІТСКЛА СЕРВІС — сайт

Статичний сайт компанії «Світскла Сервіс» (виготовлення та монтаж виробів зі скла: душові перегородки, шторки на ванну, дзеркала, дзеркала з підсвіткою, дзеркала в рамах, скляні перила, перегородки, двері).

## Структура
- `index.html` — головна сторінка
- `catalog.html` — каталог
- `assets/` — логотипи та зображення

## Деплой
Сайт статичний — збірка не потрібна.

**Vercel:** імпортувати цей репозиторій на [vercel.com/new](https://vercel.com/new). Framework Preset: **Other**, Build Command: порожнє, Output Directory: `.` (корінь).

**GitHub Pages:** Settings → Pages → Source: `main` / `/ (root)`.
